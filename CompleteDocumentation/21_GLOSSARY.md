# GLOSSARY — ResumeXpert AI

A beginner-friendly reference for technical terms used throughout the ResumeXpert AI platform and its documentation.

---

## 🔤 A

**ATS (Applicant Tracking System)**
Software used by companies to automatically filter and rank job applications. ResumeXpert AI simulates these systems to help job seekers optimize their resumes for better visibility.

**API (Application Programming Interface)**
A set of rules allowing the frontend (React) and backend (Express) to communicate. All platform features (Analysis, Interviews, Builder) rely on this bridge.

**Authentication**
The process of verifying user identity. ResumeXpert AI uses secure JWT-based authentication for both registered and guest users.

**Authorization**
Verifying user permissions. Controls access to personal data, premium templates, and administrative tools.

---

## 🔤 C

**Career OS**
The core design philosophy of ResumeXpert AI — an all-in-one "Operating System" for job seekers that integrates analysis, building, matching, and coaching into a single high-density interface.

**CORS (Cross-Origin Resource Sharing)**
A security mechanism that allows the ResumeXpert AI frontend to safely communicate with its backend server.

**CSS Variables (RGB Theme)**
A system of dynamic color tokens (e.g., `rgb(110, 86, 207)`) that enables seamless theme transitions and premium visual effects across the platform.

---

## 🔤 F

**Fallback Chain (AI)**
A reliability mechanism where ResumeXpert AI attempts to process data using Google Gemini (Primary), failing over to OpenAI or Groq if needed to ensure 100% uptime.

**Feature Marquee**
A high-density visual element on the landing page that provides an infinite-scrolling showcase of core platform modules.

---

## 🔤 G

**Gemini 2.0 Flash**
The primary AI engine powering ResumeXpert AI, known for its extreme speed and accuracy in document parsing and contextual analysis.

**Guest Demo**
A feature allowing users to instantly experience the "Career OS" with a single click, using temporary, isolated account credentials.

---

## 🔤 J

**JWT (JSON Web Token)**
A secure, encrypted string used to maintain user sessions without the need for server-side state.

---

## 🔤 L

**LLM (Large Language Model)**
The underlying technology (Gemini, GPT-3.5, LLaMA) used to "read" resumes and "talk" to users during mock interviews.

---

## 🔤 M

**MERN Stack**
The technological foundation of ResumeXpert AI: **M**ongoDB, **E**xpress.js, **R**eact, and **N**ode.js.

---

## 🔤 S

**Stacked Cards**
An interactive landing page design where feature blocks stack vertically with sticky positioning and parallax-like effects as the user scrolls.

---

## 📋 Summary

This glossary covers core technical and design terms used in the ResumeXpert AI project. Understanding these concepts is key to navigating the high-density "Career OS" architecture.
